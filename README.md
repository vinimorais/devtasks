📝 DevTasks
DevTasks é uma aplicação web moderna de gestão de tarefas, desenvolvida especificamente para o fluxo de trabalho de programadores. O projeto utiliza as tecnologias mais recentes do ecossistema React para garantir performance, escalabilidade e uma experiência de utilizador fluida.

🚀 Tecnologias Utilizadas
O projeto foi construído com uma stack robusta:

Next.js 15+ - Framework React com suporte a App Router para renderização otimizada.

TypeScript - Tipagem estática para maior segurança e produtividade no desenvolvimento.

Tailwind CSS - Estilização utilitária para uma interface rápida e responsiva.

ESLint - Padronização de código e boas práticas.

✨ Funcionalidades (Exemplos - Ajuste conforme o seu código)
[ ] Criação de Tarefas: Adicione tarefas rapidamente com categorias ou prioridades.

[ ] Dashboard Intuitivo: Visualização clara do que precisa ser feito.

[ ] Responsividade: Interface adaptada para desktop e dispositivos móveis.

🛠️ Como Executar o Projeto
Pré-requisitos
Antes de começar, certifique-se de que tem o Node.js instalado na sua máquina.

Instalação
Clone o repositório:

Bash
git clone https://github.com/vinimorais/devtasks.git
Entre na pasta do projeto:

Bash
cd devtasks
Instale as dependências:

Bash
npm install

Execução
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Abra http://localhost:3000 no seu navegador para ver o resultado.
